LED 矩陣貪食蛇機台

用一片 8x8 LED 矩陣顯示畫面、一支類比搖桿控制方向，做出最經典的貪食蛇遊戲。這是認識「矩陣掃描顯示」與「遊戲迴圈」概念最好的第一個專案。

難度★★☆☆☆
製作時間約 2-3 小時
核心零件MAX7219 + 8x8 LED 矩陣

遊戲玩法

蛇從畫面中央開始，玩家用搖桿控制上下左右移動，每吃到一個隨機出現的「食物」LED 點，蛇身就增長一格。蛇頭碰到自己的身體或畫面邊界即遊戲結束，並在矩陣上閃爍顯示分數。

所需材料

零件數量備註
Arduino Uno / Nano1相容板皆可
MAX7219 8x8 LED 矩陣模組1已內建驅動 IC，接線少
類比搖桿模組 (KY-023)1含 X/Y 電位器與按鍵
蜂鳴器（選配）1吃到食物時發出音效
杜邦線、麵包板若干

接線提醒

MAX7219 模組吃 5V，若同時接了蜂鳴器建議另外供電，避免矩陣全亮時電流不足造成 Arduino 重開機。

核心程式邏輯

安裝 LedControl 函式庫後，遊戲迴圈的核心概念是：讀搖桿方向 → 更新蛇的座標陣列 → 檢查碰撞 → 重繪矩陣。

#include <LedControl.h>
LedControl lc = LedControl(11, 13, 10, 1);

int snakeX[64], snakeY[64], length = 3;
int dirX = 1, dirY = 0;

void loop() {
  readJoystickDirection(); // 讀 A0 / A1，更新 dirX, dirY

  // 蛇身座標整體往前移動一格
  for (int i = length; i > 0; i--) {
    snakeX[i] = snakeX[i - 1];
    snakeY[i] = snakeY[i - 1];
  }
  snakeX[0] += dirX;
  snakeY[0] += dirY;

  if (hitWallOrSelf()) { showGameOver(); return; }
  if (snakeX[0] == foodX && snakeY[0] == foodY) { length++; spawnFood(); }

  redrawMatrix();
  delay(200); // 控制遊戲速度
}

延伸挑戰

  • 加入分數顯示：遊戲結束後用矩陣閃爍次數表示分數。
  • 加入難度遞增：蛇越長，delay() 越短，速度越快。
  • 換成雙色 LED 矩陣，讓蛇頭與蛇身用不同顏色顯示。
AI 協作提示

這類「座標陣列＋碰撞判定」的邏輯很適合請 AI 幫你檢查邊界條件（例如蛇頭剛好等於陣列長度時的錯誤），或是請它幫你把遊戲速度改成用 millis() 非阻塞計時，取代 delay()。詳細做法可參考 AI 協作教學。
