夾娃娃機

三軸移動機構加上夾爪升降，是七個機台裡機構最複雜的一個。重點在於用步進馬達做出精準、可重複的座標移動，以及夾爪的開合時序控制。

難度★★★★★
製作時間約 8 小時以上（含機構組裝）
核心零件步進馬達 x3 + 伺服馬達

遊戲玩法

玩家用搖桿控制夾爪在 X、Y 兩軸移動，按下按鍵後夾爪沿 Z 軸下降、閉合、再上升，並移動到出口上方放開。是否夾到娃娃取決於機構精度與夾爪閉合力道，保留一點「夾娃娃機式的隨機手感」。

所需材料

零件數量備註
Arduino Mega 25601接腳較多，建議用 Mega
28BYJ-48 步進馬達＋ULN2003 驅動板3分別驅動 X / Y / Z 軸
SG90 伺服馬達1控制夾爪開合
類比搖桿模組1控制 X / Y 移動
限位開關2-4各軸歸零用
鋁擠型 / 3D 列印機構件1 組移動平台與夾爪本體

接線重點

模組接腳Arduino 接腳
X 軸步進驅動板 IN1-4D2-D5
Y 軸步進驅動板 IN1-4D6-D9
Z 軸步進驅動板 IN1-4D10-D13
夾爪伺服馬達D44（PWM）
搖桿 VRx / VRyA0 / A1
限位開關D22-D25
歸零很重要

步進馬達沒有絕對位置概念，開機時務必先讓每一軸移動到限位開關觸發為止，作為座標原點，之後才能用「走幾步」來精準定位，否則座標會隨著使用次數累積誤差。

核心程式邏輯

#include <Stepper.h>
#include <Servo.h>
Stepper motorX(2048, 2, 3, 4, 5);
Servo claw;

enum State { IDLE, MOVING, LOWERING, GRABBING, RAISING, RETURNING };
State state = IDLE;

void loop() {
  switch (state) {
    case IDLE:
      readJoystickAndMoveXY();
      if (buttonPressed()) state = LOWERING;
      break;
    case LOWERING:
      motorZLower();
      state = GRABBING;
      break;
    case GRABBING:
      claw.write(10); // 夾爪閉合
      delay(500);
      state = RAISING;
      break;
    case RAISING:
      motorZRaise();
      state = RETURNING;
      break;
    case RETURNING:
      moveToExitAndRelease();
      state = IDLE;
      break;
  }
}

用狀態機（state machine）取代一長串 if/else，是控制多步驟機構動作最重要的觀念——每個狀態只做一件事，切換條件清清楚楚。

延伸挑戰

  • 加入限時挑戰：60 秒內夾到算成功，逾時自動歸位。
  • 用超音波感測器偵測是否成功夾到娃娃，給予不同音效回饋。
  • 把控制方式改成手機藍牙搖桿，體驗更像真的夾娃娃機店。
AI 協作提示

狀態機的狀態轉換邏輯很適合請 AI 幫你畫成流程圖（先用文字描述每個狀態與切換條件，再請 AI 整理成清單或圖示），確認邏輯沒有漏掉的分支，再回頭寫程式碼會順很多。
