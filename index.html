// BEFORE YOU START

開始之前，先備好這些

七個教學都建立在同一套基礎工具上，準備好之後就能照著任何一頁直接開工。

基本工具

硬體

Arduino Uno 或相容板一片、USB 連接線、麵包板、跳線若干、三用電表（除錯很好用）。

開發環境

軟體

電腦安裝 Arduino IDE（或 VS Code + PlatformIO），並依機台需求安裝對應的函式庫，如 MPU6050、Servo、LedControl。

建議習慣

做法

先在麵包板上把電路接好、程式燒錄成功、遊戲邏輯跑順，最後才動手做外殼與機構，避免拆了重接。
