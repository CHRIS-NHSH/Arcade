解謎密室機關

把三種不同的輸入裝置串成一個小型密室逃脫：RFID 卡片解鎖、鍵盤密碼、燈光記憶序列。這台也是我們示範「如何用 AI 協作設計關卡與寫程式」的完整案例。

難度★★★★☆
製作時間約 5-7 小時
核心零件RFID + 矩陣鍵盤 + 電磁鎖

遊戲玩法

  1. 關卡一・感應解鎖：玩家找到藏在房間裡的 RFID 卡片並感應，觸發第一道 LED 由紅轉綠。
  2. 關卡二・密碼推理：牆上的提示需要玩家推理出 4 位數密碼，用矩陣鍵盤輸入。
  3. 關卡三・燈光記憶：四顆 LED 依序閃爍一組序列，玩家要照相同順序按下對應按鈕（類似 Simon 遊戲），連續正確三輪才過關。
  4. 通關：三關全部完成後，電磁鎖釋放，蜂鳴器播放通關音效。

電磁鎖注意

電磁鎖電流較大，務必透過繼電器模組驅動，不要直接接到 Arduino 數位腳，否則會燒毀輸出腳，嚴重時損壞整片板子。

核心程式邏輯：用狀態機串接三道關卡

enum Stage { STAGE_RFID, STAGE_KEYPAD, STAGE_MEMORY, ESCAPED };
Stage stage = STAGE_RFID;

void loop() {
  switch (stage) {
    case STAGE_RFID:
      if (validCardDetected()) { setLED(1, GREEN); stage = STAGE_KEYPAD; }
      break;
    case STAGE_KEYPAD:
      if (checkKeypadCode("1937")) { setLED(2, GREEN); stage = STAGE_MEMORY; }
      break;
    case STAGE_MEMORY:
      if (playAndCheckMemorySequence()) { setLED(3, GREEN); stage = ESCAPED; }
      break;
    case ESCAPED:
      unlockDoor();
      playWinSound();
      break;
  }
}

延伸挑戰

  • 加入計時器與失敗次數限制，營造更緊張的密室氣氛。
  • 用 SD 卡模組播放語音提示，取代單純的燈光與蜂鳴器。
  • 把三關卡順序改成可設定，方便重複遊玩、更換謎題。
