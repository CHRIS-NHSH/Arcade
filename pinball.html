彈珠彈跳台

用感測器偵測彈珠滾過的位置，觸發伺服馬達彈開擋板或翻板，做出縮小版的彈珠台機構。這是第一個真正結合「感測＋機構動作」的實體機台。

難度★★★★☆
製作時間約 5-6 小時（含機構）
核心零件紅外線感測 + 伺服馬達

遊戲玩法

彈珠從斜面滾入木製或壓克力台面，滾過不同位置的紅外線感測點會得到不同分數，並觸發對應的伺服彈板把彈珠彈向另一個方向，增加隨機性與趣味。彈珠最終落入底部的計分洞。

所需材料

零件數量備註
Arduino Uno1接腳較多，建議用 Uno 而非 Nano
紅外線反射式感測模組3-5偵測彈珠是否經過
SG90 伺服馬達2-3驅動彈板 / 翻板
七段顯示器（TM1637）1顯示分數
木板 / 壓克力板、斜面軌道1 組台面機構

接線重點

模組接腳Arduino 接腳
紅外線感測 1 / 2 / 3D2 / D3 / D4
伺服馬達 1 / 2D9 / D10（PWM 腳）
TM1637 CLK / DIOD6 / D7
供電提醒

伺服馬達啟動瞬間電流較大，若同時驅動兩顆以上，建議用獨立的 5V 電源供應伺服馬達，只共用「地線（GND）」與 Arduino 相連，避免電壓瞬降讓 Arduino 重開機。

核心程式邏輯

#include <Servo.h>
Servo flipper1, flipper2;
int score = 0;

void loop() {
  if (digitalRead(2) == LOW) {   // 彈珠經過感測點 1
    score += 10;
    fireFlipper(flipper1);
  }
  if (digitalRead(3) == LOW) {
    score += 20;
    fireFlipper(flipper2);
  }
  updateScoreDisplay(score);
}

void fireFlipper(Servo &s) {
  s.write(120);   // 彈開
  delay(150);
  s.write(30);    // 收回原位
}

延伸挑戰

  • 加入「連續命中加倍」機制：短時間內連續觸發感測器分數翻倍。
  • 用電磁閥取代伺服馬達做出真正的彈射力道。
  • 加入音效模組，命中不同感測點播放不同音效。
AI 協作提示

這個機台的難點常常在機構而不是程式——彈板角度、彈珠軌道斜度需要多次試誤。可以把你想要的手感（彈得多遠、多快收回）描述給 AI，請它幫你估算 delay() 時間與伺服角度的組合作為起點，再實測調整。
